# Landing Page Luz da Moda

Uma landing page responsiva para a loja de moda feminina Luz da Moda, localizada em Dois Vizinhos - PR.

## Tecnologias

- Vite
- JavaScript Vanilla
- CSS3 (Mobile-First)

## Funcionalidades

- Hero Section com chamada para ação
- Catálogo de produtos com filtros
- Depoimentos de clientes
- Integração com WhatsApp e Instagram
- Design responsivo

## Como executar

1. Instale as dependências: `npm install`
2. Execute o servidor de desenvolvimento: `npm run dev`
3. Abra o navegador em `http://localhost:5173`

## Painel Administrativo

Para acessar o painel de administração e atualizar produtos:
- Acesse `http://localhost:5173/?admin`
- Faça login com usuário: `Admin` e senha: `Admin`
- Adicione, edite ou remova produtos
- Salve as alterações no navegador local
- Exporte/importe dados em JSON para backup

## Responsividade

- **Mobile-first:** Otimizado para dispositivos móveis
- **Tablet:** Layout adaptado para tablets (720px+)
- **Desktop:** Layout expandido para telas grandes (1024px+)
- **Ultra-wide:** Suporte para telas muito largas (1440px+)

## Novidades

- Layout melhorado com localização da loja e entrega condicional.
- Catálogo organizado por peça, com descrições e tamanhos visíveis.
- Foco em conversão via WhatsApp e agendamento local.

## Imagens

As imagens usadas no catálogo podem ser atualizadas usando os arquivos da pasta `Catalogo`.
